# capture-ur-day
RUNTEQ卒制アプリ | AI添削付き英語ジャーナリングアプリ

---

# サービス概要

**capture-ur-day** は  英語学習者が日々の出来事や感情を英語で書き、  
AIの添削を通して **「間違いを恐れずアウトプットできる環境」** を提供する **AI添削付き英語ジャーナリングアプリ**です。

英語を学習している人が

- 自分の英語の成長を感じる
- 英語で自分の感情を表現できる
- アウトプットの習慣を作る
- 継続することで自信の構築、アウトプットに対する心理的ハードルを下げる

ことを目的としています。

---

# このアイデアが生まれた背景

## きっかけとなった体験

英語学習を続ける中で、次のような経験をしました。

- リスニングはできるし、単語は覚えられる
- しかし **英語を話す・書く場面になると言葉が出てこない**

原因を振り返ると

- **間違えることが恥ずかしい**
- **アウトプットする機会が少ない**

という心理的なブロックがありました。

語学学習では  「間違いながらアウトプットすること」が重要ですが、

実際には
**恥ずかしさ → アウトプット回避 → 上達しない**
という状態になっていました。


そこで

**AIとの対話なら、間違いを気にせずアウトプットできるのではないか**

と考え、このサービスを作ろうと思いました。

---

## なぜこの問題が気になったのか

私はTOEIC965・英検準1級を取得していますが、実際のスピーキングは得意とは言えません。

原因を考えると

- アウトプット回数が少ない
- 間違うことへの恐怖
- 失敗するときに感じる「恥ずかしさ」「自分へのがっかり感」を避ける心理

がありました。

この経験から

**「間違いを気にしなくていい環境」**　**「小さな成功体験を積み重ねる環境」**

が重要だと感じました。

AIの添削を通して

- 間違いを修正できる
- 正しい表現を学べる
- 次のアウトプットに自信が持てる

という **成長サイクル** を作りたいと考えています。

---

# 解決したい課題

英語学習者には次の問題があります。

- 英語を間違えるのが恥ずかしい
- アウトプットの機会が少ない
- 自分の間違いを振り返る仕組みがない

その結果

- 英語を使う自信が持てない
- 成長を実感できない
- 間違いを正せないままになってしまう

という問題が起きます。

このアプリでは

**英語アウトプット → AI添削 → 振り返り**

のサイクルを作ることで

**英語を使う心理的ハードルを下げること**を目的としています。

AIの添削によって自分の間違いを修正し小さな成功体験を積み重ねることで、

英語を使うことへの自信を育て、新しいことに挑戦するきっかけになることを目指しています。

# 懸念点

### APIコスト

AI APIの使用回数が増えるとコストが高くなる可能性があります。

対策

- 利用回数制限
- お試し期間
- 将来的な有料プラン

### 継続率

日記を書く習慣が続かない可能性があります。

対策

- リマインダー通知
- 週1回振り返り通知
